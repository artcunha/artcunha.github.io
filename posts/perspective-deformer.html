Perspective Deformer


The idea with this deformer is to give animators the control of that foreshortening on object basis, based on a user-defined distance. This also provides a non-destructive way of posing to camera, which can potentially enhance workflows with downstream departments like layout and sim.

Pushing non-linear perspective

Foreshortening is a core drawing fundamental - it describes how quickly objects distort and fall into perspective. This doesn’t always follows linear rules and it’s often pushed by artists.

Model View Projections

The actual technique is straightforward, instead of multiplying the vertex position by the MVP matrix like in a regular vertex shader, we calculate the delta of the matrix by a new one based on the  deformer's custom Focal Length input. This is also multiplied by a float ramp mapped to the distance range so that we can change the Focal Length in Z, effectively allowing us to "bend" the perspective lines.
